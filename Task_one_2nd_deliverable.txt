GraphQL Queries:
1. Assume the client will use the header x-hasura-artist-id for identifying the Artist
   Artists should not be allowed to access albums that do not belong to them
->
Query:
~~~~~~~~~~~~~~~~~~~~~~~
query MyQuery {
  album {
    album_id
    artist_id
    title
  }
}

Request Headers:
~~~~~~~~~~~~~~~~~~~~~~~
content-type: application/json
x-hasura-admin-secret: Pinkgreen
x-hasura-artist-id: 1
x-hasura-role: user

Query Response:
~~~~~~~~~~~~~~~~~~~~~~~
{
  "data": {
    "album": [
      {
        "album_id": 1,
        "artist_id": 1,
        "title": "For Those About To Rock We Salute You"
      },
      {
        "album_id": 4,
        "artist_id": 1,
        "title": "Let There Be Rock"
      }
    ]
  }
}
#################################################################################

2. How many artists are in the database?
->
Query:
~~~~~~~~~~~~~~~~~~~~~~~
query MyQuery {
  artist_aggregate {
    aggregate {
      count(distinct: true)
    }
  }
}

Request Headers:
~~~~~~~~~~~~~~~~~~~~~~~
content-type: application/json
x-hasura-admin-secret: Pinkgreen

Query Response:
~~~~~~~~~~~~~~~~~~~~~~~
{
  "data": {
    "artist_aggregate": {
      "aggregate": {
        "count": 275
      }
    }
  }
}
#################################################################################

3. List the first track of every album by every artist in ascending order
->
Query:
~~~~~~~~~~~~~~~~~~~~~~~
query MyQuery {
  album(order_by: {title: asc, artist: {artist_id: asc, albums_aggregate: {}}}) {
    album_id
    title
    artist {
      name
    }
    tracks(order_by: {track_id: asc, media_type: {}, album: {artist: {albums_aggregate: {}}}, unit_price: asc}, limit: 1) {
